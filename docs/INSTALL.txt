medialog.iconpicker Installation
--------------------------------

To install medialog.iconpicker using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``medialog.iconpicker`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        medialog.iconpicker
       
* Re-run buildout, e.g. with:

    $ ./bin/buildout
